How to migrate from one server to another?

For example you have to move your current admin container to another server. Besides of pulling and installer docker image from Dockerhub you need to migrate mongodb data (in case you want to save it). By default mongo stores it's data in /var/lib/just/mongodb folder on your server. So you just need to stop an admin container on the new server, move mentioned folder to the new server and start the container.
